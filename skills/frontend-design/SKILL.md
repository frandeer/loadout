# frontend-design

이 모듈은 AI 느낌의 천편일률적이고 뻔한 디자인을 피하고, Black-ops 군사 시뮬레이션 및 Anomaly Tactical Console 스타일을 추구하여 사용자에게 깊은 인상과 높은 사용성을 제공하는 프런트엔드 UI 설계 스킬입니다.

## 주요 지침
- **Premium Aesthetics**: 어두운 그래파이트 패널, 테마별 라인 발광, 미세 스캔라인 및 격자망 레이아웃 적용.
- **Micro-Animations**: 요소 위에 스캔라인이 한 번 지나가는 등의 절제된 피드백 모션.
- **Color Tokens**: 07-card-design.md의 컬러 토큰(#05080A, #101514, #18201D, #D49A2A, #38D6C6, #39D98A, #8D5CFF, #D45A3C 등) 엄격 준수.
