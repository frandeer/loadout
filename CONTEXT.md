# Loadout

Claude Code의 skill·agent·MCP·memory를 게임 카드처럼 모아 비교하고 `~/.claude`에
장착해 쓰는 개인 관리 시스템. 이 문서는 도메인 용어집이며 구현 결정은 담지 않는다.

## Language

### 자산의 로드 상태

**로드아웃 장착 (Active loadout)**:
사용자가 Loadout UI에서 *의도적으로* `~/.claude`에 연결(junction/copy)한 자산. "지금 내가 켠 것".
_Avoid_: equipped(코드 한정), 활성

**설치 베이스 (Install base / Ambient)**:
플러그인·직접 설치로 `~/.claude`에 *이미* 존재하는 자산. 장착과 무관하게 컨텍스트에 로드된다.
"내가 켠 건 아니지만 어차피 로드되는 것".
_Avoid_: 앰비언트(영문 음차), 상주(맥락에 따라만)

**보관 (Vault)**:
`~/.claude`에서 내려 컨텍스트에서 빠진 상태. 카탈로그에는 남지만 로드되지 않는다.
_Avoid_: 비활성, 삭제(삭제는 카탈로그에서까지 제거)

### 컨텍스트 비용

**상시 비용 (Always-on cost / descCost)**:
자산이 켜져 있기만 해도 매 턴 컨텍스트에 들어가는 비용(주로 description/frontmatter). 작다.

**호출 시 비용 (On-demand cost / cost)**:
자산을 실제 호출(로드)할 때 본문이 들어가는 비용. 크다. `거대 자산(oversized)` 판정의 기준.
_Avoid_: 본문 비용(UI 라벨로는 가능)

### 화면

**관제탑 (Control tower / Dashboard)**:
착지 화면. 사용자의 1순위 의도는 **"지금 내 Claude 상태 = 무엇이 켜져 있고 / 컨텍스트를
얼마나 먹고 / 뭘 꺼야 하나"**(A). 발견·재미는 보조. 3초 스캔용 *다이제스트*.

**무기고 지도 (Arsenal map)**:
전(前) "그래프" 탭을 대체하는 도메인 군집 트리맵. 모집단은 *전체 카탈로그*, 블록 = `category`,
블록 면적 = 자산 *개수*, 타일 색 = 로드 상태, 무게(토큰)는 색/주석으로 얹는다.
관제탑이 다이제스트라면 지도는 펼쳐 탐색하는 *캔버스*. 관계는 *군집 멤버십*으로 표현하고
pairwise 엣지(동반/유사)는 그리지 않는다.
_Avoid_: 그래프(폐기된 force node-link을 가리킴), 온톨로지
